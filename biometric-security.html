Biometric Access Controlled Automotive Security System

Enhancing vehicle security with advanced biometric authentication technology.

Project Overview

This project replaces traditional keys with a biometric fingerprint recognition system, providing a secure and user-friendly solution for automotive access and control.

Features

  • Fingerprint authentication for secure vehicle access.
  • Easy enrollment and verification of fingerprints.
  • Compact and efficient circuit design.

How It Works

  • Enrollment: Assign unique IDs to fingerprints and store them securely.
  • Verification: Match fingerprints against stored templates to allow or deny access.

Technical Details

  • Organization: Alcom Extrusions Pvt Ltd, Hyderabad
  • Role: Lead Developer
  • Team Size: 3
  • Tools & Technologies: Arduino UNO, AS608 fingerprint sensor, Adafruit Fingerprint Sensor Library, Circuit Prototyping.

Problem Statement

Traditional automotive security systems are prone to unauthorized access and key duplication. The project aims to enhance security by replacing keys with biometric authentication.

Contributions

  • Designed and implemented a compact and efficient biometric system.
  • Developed firmware for secure fingerprint enrollment and verification.
  • Optimized circuit design to ensure reliability and efficiency.
